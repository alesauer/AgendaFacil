# Roteiro de Vídeo — AgendaFácil

> **Formato:** Vídeo demonstrativo comercial (60s a 90s)
> **Público-alvo:** Donos de barbearia no Brasil
> **Tom:** Direto, profissional, acolhedor
> **Plataforma:** YouTube / Instagram Reels / TikTok / Site institucional

## ANEXO: Sugestões de produção para ElevenLabs

| Parâmetro | Valor sugerido |
|---|---|
| **Voz** | Modelo masculino, tom firme e acolhedor (ex: "Fábio", "Antônio") |
| **Velocidade** | 1.0x a 1.05x (sem pressa, sem arrastar) |
| **Pausas** | Inserir pausa de 0.3s a 0.5s entre cenas |
| **Estilo** | Narração informativa, levemente entusiástica (sem exageros) |
| **Background** | Música instrumental leve, ritmo médio (lo-fi ou corporate) |
| **Estabilização** | Usar ferramenta de *audio cleaning* se disponível |
